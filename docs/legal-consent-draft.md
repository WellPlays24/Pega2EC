# Borrador Funcional De Consentimiento

Este texto es una base funcional para el registro. Debe ser revisado por un abogado antes de publicarse.

## Aceptacion De Registro

Al crear una cuenta en Pega2EC, el usuario declara y acepta que:

1. es mayor de 18 anos
2. la informacion entregada es real, actual y verificable
3. la plataforma puede solicitar y almacenar foto de cedula, foto de perfil y otros datos necesarios para validacion manual
4. la cuenta no podra activarse hasta ser revisada y aprobada por un administrador
5. la plataforma podra rechazar la solicitud si detecta inconsistencias, suplantacion, datos falsos o incumplimiento de politicas

## Tratamiento De Datos

El usuario acepta que:

1. ciertos datos visibles del perfil podran mostrarse a otros usuarios autenticados, como alias, edad, provincia, canton, descripcion, intereses y fotos aprobadas
2. la ubicacion exacta podra ser almacenada para filtros internos, pero no sera publicada de forma visible a otros usuarios
3. los administradores autorizados podran acceder a chats, imagenes, cedulas, evidencias y registros internos cuando sea necesario para moderacion, auditoria, seguridad o cumplimiento

## Venta De Datos Personales Configurados Por El Usuario

El usuario acepta que:

1. solo pondra a la venta informacion que haya habilitado expresamente dentro de su perfil
2. los precios de desbloqueo seran definidos por la plataforma
3. cuando otro usuario compre acceso a un dato habilitado, este acceso sera temporal segun las reglas vigentes de la plataforma
4. la plataforma registrara cada compra, desbloqueo y acceso para fines de auditoria y soporte

## Puntos, Pagos Y Retiros

El usuario acepta que:

1. todas las interacciones pagadas funcionan mediante `Pega2Points`
2. los puntos no son reembolsables, salvo casos excepcionales definidos por la plataforma como cancelacion de eventos
3. los retiros de ganancias solo podran solicitarse conforme a las reglas vigentes y mediante datos bancarios validos del titular
4. cualquier fraude, chargeback o uso indebido puede causar suspension o baneo permanente

## Moderacion Y Sanciones

El usuario acepta que:

1. cualquier reporte debe sustentarse con evidencia
2. la plataforma podra aplicar advertencias, suspensiones temporales, desbaneo pagado o baneos permanentes segun la gravedad del caso
3. ciertos comportamientos, como fraude, documento falso o envio de contenido sexual no consentido, pueden generar baneo permanente sin posibilidad de recuperacion

## Eliminacion De Cuenta

La eliminacion de cuenta se tramita por solicitud para no afectar compras, auditorias, historiales y obligaciones pendientes.
